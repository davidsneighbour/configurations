## Usage

Install as dev-dependency:

```shell
npm install -D dnb-hugo/browserslist-config-dnb
```

Then add as extension to the `browserslist` parameter in package.json.

```json
{
  "browserslist": ["@dnb-hugo/extends browserslist-config-dnb"]
}
```
