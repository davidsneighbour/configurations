# ESLint

Add .eslintrc with the following content to your project:

```json
{
  "extends": "@davidsneighbour/eslint-config"
}
```

or in `package.json`:

```json
{
  "eslintConfig": {
    "extends": "@davidsneighbour/eslint-config"
  }
}
```

Write better code.
