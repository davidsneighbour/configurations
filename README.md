## dnb-configurations

Just a repo that unites my development processes across the configuration packages within dnb-hugo. See [dnb-hugo on npmjs.com](https://www.npmjs.com/search?q=config%20dnb-hugo) for more details. 

### All configuration packages

| Package | Notes |
|---|---|
| [browserslist-config](https://github.com/dnb-hugo/browserslist-config) | Browserslist configuration used in dnb-hugo projects. |
| [commitlint-config](https://github.com/dnb-hugo/commitlint-config) | [Commitlint](https://github.com/conventional-changelog/commitlint) configuration used in dnb-hugo projects. |
| [eslint-config](https://github.com/dnb-hugo/eslint-config) | ESLint configuration used in dnb-hugo projects. |
| [remark-config](https://github.com/dnb-hugo/remark-config) | Remark configuration used in dnb-hugo projects. |
| [standard-version-config](https://github.com/dnb-hugo/standard-version-config) | Standard Version configuration used in dnb-hugo projects. |
| [stylelint-config](https://github.com/dnb-hugo/stylelint-config) | Stylelint configuration used in dnb-hugo projects. |
| [textlint-config](https://github.com/dnb-hugo/textlint-config) | Textlint configuration used in dnb-hugo projects. |
